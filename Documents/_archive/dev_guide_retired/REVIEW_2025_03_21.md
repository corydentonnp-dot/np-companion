## 1. CRITICAL — Fix Immediately

### 1.1 Hardcoded Credentials in config.py

**File:** `config.py` lines 30–104  
**Issue:** Every `os.getenv()` call has a hardcoded fallback with real credentials:

| Line | Credential | Risk |
|------|-----------|------|
| 30 | `SECRET_KEY` (Flask signing key) | Session forgery, CSRF bypass |
| 45 | `AC_LOGIN_PASSWORD` ("ASDqwe123") | Amazing Charts login exposed |
| 71 | `OPENFDA_API_KEY` | FDA API abuse |
| 77–78 | `LOINC_USERNAME` / `LOINC_PASSWORD` | LOINC account compromise |
| 81 | `UMLS_API_KEY` | UMLS access abuse |
| 84–85 | `VIIS_USERNAME` / `VIIS_PASSWORD` | VA immunization system |
| 88–89 | `PDMP_EMAIL` / `PDMP_PASSWORD` | Prescription monitoring access |
| 103–104 | `PUSHOVER_USER_KEY` / `PUSHOVER_API_TOKEN` | Push notification hijack |

**Resolution:**
1. Remove all default values from `os.getenv()` calls — make them raise on missing
2. Rotate ALL exposed credentials immediately (they're compromised once committed)
3. Verify `config.py` is in `.gitignore` (it is, but verify the git history)
4. Use `config.example.py` pattern with empty placeholders only

### 1.2 Traceback Exposed to Users in HTTP 500 Responses

**File:** `app/__init__.py` lines 188–195  
**Issue:** Full Python traceback passed to error template and rendered to the browser:

```python
tb = traceback.format_exc()
return render_template('errors/500.html', error=error, traceback=tb), 500
```

This leaks internal file paths, SQL queries, variable names, and stack frames.

**Resolution:** Only pass `traceback=tb` when `app.debug is True`. In production, pass `traceback=None`.

```javascript
html += '<strong>' + p.name + '</strong>'  // XSS if name contains <script>
```

### 1.4 Pickle Deserialization in Scrapers

**File:** `scrapers/netpractice.py` ~line 234  
**Issue:** Cookie files loaded with `pickle.load()` — arbitrary code execution if file tampered:

```python
with open(self.cookie_file, 'rb') as f:
```

### 1.5 PHI in Filenames — Clinical Summary XML

**File:** `agent/clinical_summary_parser.py` ~line 153  
**Issue:** XML export files contain MRN in their filename:

```python
pattern = f'ClinicalSummary_PatientId_{mrn}_'
```

**Also:** Screenshots in `data/screenshots/` saved with no access control, no encryption, and persist indefinitely.

**Resolution:**
1. Hash MRN before using in filenames: `sha256(mrn.encode()).hexdigest()[:12]`
2. Set restrictive file permissions on `data/screenshots/` and `data/clinical_summaries/`
3. Implement the scheduled deletion job (see 1.6)

### 1.8 ZIP Extraction Without Path Validation

**File:** `utils/updater.py` lines 70–95  
**Issue:** `shutil.copytree()` copies extracted ZIP content to internal directory without validating for path traversal (e.g., `../../etc/passwd`).

**Resolution:** Validate all extracted paths are within the target directory before copying:

```python
for member in archive.namelist():
    target = os.path.realpath(os.path.join(dest, member))
    if not target.startswith(os.path.realpath(dest)):
        raise ValueError(f"Path traversal detected: {member}")
```

### 2.4 XML Watcher Can Leak Observer Instances

**File:** `agent/clinical_summary_parser.py` lines 678–701  
**Issue:** `start_xml_watcher()` assigns a new `Observer()` to `_watcher_observer` without stopping the previous one. Calling it twice leaks the first observer thread.

```python
def start_xml_watcher(user_id, user=None):
    global _watcher_observer
    if _watcher_observer is not None:
        _watcher_observer.stop()
        _watcher_observer.join(timeout=5)
    # ... create new observer
```

### 3.1 No Rate Limiting on Any Endpoint

**Impact:** Any authenticated user can spam endpoints indefinitely.

**Priority targets:**

| Endpoint | Risk |
|----------|------|
| `/api/medref/lookup` | External RxNorm/OpenFDA API calls |

### 3.9 Agent Automation — Rigid Sleep-Based Timing

~15 locations use `time.sleep()` with hardcoded values (0.2s–15s). If Amazing Charts is slow, automation fails despite the element being present.

**Resolution:** Replace `time.sleep(N)` with polling loops:

```python
def wait_for_text(target, timeout=10, interval=0.5):
    deadline = time.time() + timeout
    while time.time() < deadline:
        if ocr_find_text(target):
            return True
        time.sleep(interval)
    return False
```

### Resilience Principles Going Forward

1. **Never trust input** — Validate at every route boundary (type, length, range, format)
2. **Fail loudly** — Replace `except: pass` with `except SpecificError: logger.warning(...)` 
3. **Guard DOM access** — Every `getElementById()` result gets a null check before use
4. **Lock shared state** — Any module-level mutable variable accessed from scheduler jobs needs a `threading.Lock()`
5. **Encrypt at rest** — PHI in filenames, screenshots, and XML exports must be hashed or encrypted
6. **Limit blast radius** — Rate limit, paginate, and bounds-check every user-facing input
7. **Cascade correctly** — Every parent-child relationship in models must have `cascade="all, delete-orphan"`
8. **Test the edges** — Add tests for: empty lists, None values, zero denominators, max-length strings, unknown payer types
