Update Instructions

  1. On your personal PC, run python build.py to create a new build zip.
  2. Transfer the zip to this PC:
    • Google Drive: Upload zip → download on this PC
    • USB drive: Copy zip to USB → plug in here
    • Email: Attach and download (small updates only)
  3. Enter the folder path above (e.g. C:\Users\coryd\Downloads or E:\).
  4. Click Scan Folder — the newest version will appear.
  5. Click Apply Update → the app extracts and replaces code files.
  6. Click Restart → the app relaunches with the new version.

Your data is safe. Updates never touch: database, config.py, browsers, or tesseract.
