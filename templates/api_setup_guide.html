API Setup Guide

For Providers & IT Staff

NP Companion uses free, publicly available government APIs to power its clinical intelligence features. Most APIs require no setup at all. A few optional registrations unlock higher rate limits or additional features. This guide walks through each.

{# ---- Optional: OpenFDA ---- #}
🔑 Optional: OpenFDA API Key (Recommended)
What it unlocks: Higher rate limits for drug label lookups, recall checks, and adverse event queries.
Without key: 40 requests/minute  |  With key: 240 requests/minute
Features affected: Drug Safety Panel, Drug Recall Alerts, Morning Briefing recall check

Setup Instructions:

  1. Go to https://open.fda.gov/apis/authentication/
  2. Enter your email address in the "Your Email" field
  3. Click "Get API Key"
  4. Check your email — the API key will arrive within minutes
  5. Give the key to your system administrator to add to NP Companion's configuration
Cost: Free  |  Time: ~2 minutes  |  Requires: Email address only
{# ---- Optional: NCBI / PubMed ---- #}
🔑 Optional: NCBI / PubMed API Key
What it unlocks: Higher rate limits for PubMed clinical guideline searches.
Without key: 3 requests/second  |  With key: 10 requests/second
Features affected: Clinical Guidelines widget, Morning Briefing guideline pre-load

Setup Instructions:

  1. Go to https://www.ncbi.nlm.nih.gov/account/
  2. Create a free NCBI account or sign in
  3. After logging in, go to Settings (top right menu)
  4. Scroll down to "API Key Management"
  5. Click "Create an API Key"
  6. Copy the generated key
  7. Give the key to your system administrator
Cost: Free  |  Time: ~5 minutes  |  Requires: NCBI account (email + password)
{# ---- Optional: LOINC ---- #}
🔑 Optional: LOINC Account (For Lab Reference Ranges)
What it unlocks: Lab test identification, reference ranges, panel grouping via the LOINC FHIR API.
Features affected: Lab Tracker reference ranges, Abnormal Lab Interpretation Assistant (future)

Setup Instructions:

  1. Go to https://loinc.org/get-started/
  2. Click "Create a free account"
  3. Fill in the registration form (name, email, organization, intended use)
  4. For "intended use" select "Clinical/Healthcare"
  5. Confirm your email and set a password
  6. Your LOINC username and password are used for API access
Cost: Free  |  Time: ~5 minutes  |  Requires: Email, name, organization
{# ---- Optional: UMLS ---- #}
🔑 Optional: UMLS License (For Advanced Terminology)
What it unlocks: Cross-terminology mapping (SNOMED ↔ ICD-10 ↔ RxNorm ↔ LOINC), differential diagnosis support.
Features affected: Coding Suggester code pairing, Differential Diagnosis Widget (future)

Setup Instructions:

  1. Go to https://uts.nlm.nih.gov/uts/
  2. Click "Sign In" — you can use login.gov, Google, or Microsoft
  3. After authentication, NLM will show a license request form
  4. Complete the form — indicate clinical use
  5. Wait for NLM review (up to 3 business days)
  6. Once approved, go to "My Profile" → "UMLS Licensee Profile" to find your API key
Cost: Free  |  Time: 5 min registration + up to 3 business days review  |  Requires: login.gov / Google / Microsoft account
{# ---- HIPAA Note ---- #}
HIPAA Compliance: NP Companion NEVER sends patient-identifiable information to any external API. All API requests contain only clinical vocabulary terms (drug names, ICD-10 codes, LOINC codes). No patient names, MRNs, dates of birth, or other identifiers leave the local system. All data enrichment happens locally after API responses are cached.
